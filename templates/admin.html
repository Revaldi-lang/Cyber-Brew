💡 Info Uji Keamanan (RBAC & Cookie Security): {% if not SECURITY_MODE %}

RBAC Lemah & Cookies Tidak Aman:

  • RBAC Bypass: Anda dapat mengakses halaman ini meskipun menggunakan akun user biasa dengan cara memanipulasi cookie di browser Anda: tambahkan cookie session_role=admin atau ubah session_user=admin.
  • Pencurian Cookie via XSS: Karena cookie session tidak memiliki flag HttpOnly, JavaScript penyerang dapat membaca cookie ini. Cobalah menyisipkan ulasan XSS di halaman Menu, lalu buka halaman Menu tersebut menggunakan akun Admin. Periksa bagian Cookie Logs di atas untuk melihat cookie Admin yang tercuri!
{% else %}

RBAC Ketat & Cookies Terlindungi:

  • RBAC Aman: Server memvalidasi data user dan role langsung dari session terenkripsi di sisi server. Memanipulasi cookie di browser client tidak akan memberikan hak akses admin.
  • HttpOnly Cookie Enforced: Cookie ditandai dengan flag HttpOnly, yang membuat browser melarang JavaScript untuk membacanya (document.cookie kosong). Serangan XSS tidak akan bisa mencuri cookie session Anda.
